# Encryptor

## Project requirements

Age is a simple, modern and secure file encryption tool, format, and Go library.

The project should be a GUI wrapper for age. It should allow non-technical
people to easily share secrets using a secure method. The age is a great to for
that but it requires technical knowledge and it requires user to use cli.

The project should allow user to create pair of private and public key used
for encryption. It should allow users to easily share their public key.
It should allow to create contacts to easily saved other people public keys.
It should allow to encrypt files using password but preferably using
public keys. It should also allow to decrypt files using user's private key.

The interface should be user friendly and easy to navigate. It should also allow
for keyboard only navigation.
